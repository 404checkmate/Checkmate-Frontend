## 유형 체계

### A축 — 체스 기물 유형 (6종)
| 유형 | 코드 | 설명 |
|------|------|------|
| 폰 | pawn | 여유 힐링형, 의견 없이 따라감 |
| 룩 | rook | 감성 기록형, 핵심만 효율적으로 |
| 나이트 | knight | 액티비티형, 즉흥·탐험 |
| 비숍 | bishop | 문화 탐험형, 계획 사수 |
| 퀸 | queen | 올라운더형, 모든 변수 대비 |
| 킹 | king | 리더형, 아낌없이 해결 |

### B축 — 선호 여행 스타일 (8종)
| 유형 | 설명 |
|------|------|

| 문화예술 | 미술관·박물관·공연 |
| 힐링 | 휴식·회복·여유 |
| 포토스팟 | 감성 사진·인증샷 |
| 쇼핑 | 쇼핑·기념품 |
| 명소방문 | 랜드마크·관광지 |
| 나이트라이프 | 밤문화·클럽·바 |
| 액티비티 | 번지점프·래프팅 등 스릴 활동 |

| B | 숙소랑 교통편만 잡으면 끝! 나머지 자잘한 건 굳이 미리 안 정해도 돼 | [룩] | 핵심만, 군더더기 거부 |
